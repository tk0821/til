## インストール

https://gradle.org/releases/

1. 上記リンクのDownloadとある行のbinary-onlyかcompleteからダウンロードする。
2. zipを展開し、配置したいディレクトリに配置する

```
C:\Gradle\gradle-8.4-all
```

3. 環境変数にbinを追加する

```
C:\Gradle\gradle-8.4-all\gradle-8.4\bin
```

4. ターミナルでgradle -v
```
> gradle -v
```
